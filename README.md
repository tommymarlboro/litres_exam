Litres exam test
=============================

This is solution to the test task for Litres Python developer.

## Prerequisites
For correct execution of the script, you must have **Python >= 3.6** only

## How to run script
1. Clone the repository via `https://github.com/tommymarlboro/litres_exam.git`
2. Drill down into the folder `cd litres_exam`
3. Run the script `python3 csv_parser.py`
4. Enter path to 'Orders.csv' eg `C:\Users\Documents`
